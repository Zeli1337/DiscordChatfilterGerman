Discord Bot mit einigen API anbindungen so wie zB einer news search api, quote api, math api, lyrics api und einem Verwarnungsystem. 

Bei dem Projekt handelt es sich um ein altes Projekt meinerseits, welches ich vor 1 1/2 Jahren angefangen habe. Je nachdem wie ich Zeit und Lust habe werde ich zu diesem Projekt Funktionalitäten hinzufügen. 
Bisher verfügt der Bot über eine Verwarnungsfunktion, welche anspringt bei diversen selbst festzulegenden Begriffen. Nach x Verwarnungen lassen sich Funktionen wie ein Timeout, Mute, Kick, und einen Ban konfigurieren. 

Für die Verwendung würde ich replit empfehlen, jedoch ist hier zu bedenken das Replit in der Vergangenheit opfer diverser Databreaches wurde und somit Discordtokens bekannt wurden. 
Um den Bot mit Replit online zuhalten würde ich UptimeRobot empfehlen um die beigelegte Flask website zu pingen. Somit lassen sich die Serverkosten sparen.
